# 长期记忆注入说明

普通对话上下文只允许注入用户可见记忆，包括 `preference`、`constraint`、`correction`、`style_preference`、`operation_preference`、`file_impression`。

内部工具策略只允许工具路由器/执行器读取。`tool_success_profile` 可以帮助选择工具或调整执行顺序，但不要进入普通对话提示词，也不要在面向用户的回复中展示。

不要把内部工具名、执行路线或成功率统计当成用户偏好展示。需要解释操作时，只说明当前任务中用户能理解的行为和结果，不暴露长期内部统计。

当用户可见记忆与当前明确指令冲突时，优先遵守当前明确指令；必要时用简短自然的话确认，不要展开内部记忆系统细节。
