# P1 逻辑问题修复实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复审查中确认的 7 项 P1，确保压缩失败不丢历史、停止后不启动新工具、Office 操作不误改文件且 Open XML 修改保留既有结构，并让 CI 使用支持 `node:sqlite` 的 Node 版本。

**Architecture:** 修复保持现有分层：Provider 负责把请求失败表达为异常，AgentLoop 负责中断屏障，Office COM helper 负责目标解析，Open XML 模块负责局部 XML 合并。每项先增加能够稳定复现缺陷的测试，再做最小实现，不引入新的 XML 或状态管理依赖。

**Tech Stack:** TypeScript、Vitest、Electron、React/Zustand、JSZip、PowerShell、GitHub Actions。
