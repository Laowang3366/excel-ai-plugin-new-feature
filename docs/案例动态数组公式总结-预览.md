# 案例动态数组公式总结（预览）

## 提取口径

- 来源：`案例.zip` 中 15 个 `.xlsx` 工作簿。
- 只保留工作表 XML 中明确标记为 `t="array"` 的公式。
- 共提取 20 个数组公式节点；普通公式、共享填充公式和静态结果均已排除。
- 以下公式仅去除了文件内部的 `_xlfn`、`_xlpm`、`_xleta` 前缀，没有改变原公式逻辑。
- 工作簿中两处公式使用未定义名称 `x` 作为不满足条件时的占位值；另有一处使用 WPS 的 `REGEXP`。正式入库时需要单独标注环境和替代写法。

## 1. 插入字符串

位置：`Sheet1!B9`，数组范围 `B9`

```excel
=LET(fx,LAMBDA(str,arr,REDUCE(str,SEQUENCE(ROWS(arr)),LAMBDA(x,y,SUBSTITUTE(x,"{"&y&"}",INDEX(arr,y)))))),fx(XLOOKUP(A9,A2:A7,B2:B7),{777;888;999}))
```

解题结构：先用 `XLOOKUP` 取得带 `{1}`、`{2}`、`{3}` 占位符的模板；再用 `SEQUENCE` 生成替换序号，`REDUCE` 逐轮把对应参数写入模板。它虽然最终返回单个文本，但内部按数组参数迭代，文件中按数组公式保存。

可泛化方法：模板文本 + 参数数组 -> 序号驱动的逐次替换。

## 2. 分表求和

位置：`汇总!G2`，溢出范围 `G2:J140`

```excel
=LET(data,VSTACK(盛兴:天信!B2:E140),n,ROWS(A2:A140),rCount,SEQUENCE(ROWS(data))-1,rIndex,MOD(rCount,n)+1,DROP(GROUPBY(rIndex,data,SUM,0,0),,1))
```

解题结构：`VSTACK` 把多张同结构工作表纵向堆叠；用“堆叠后行号对单表行数取模”恢复原始行位置；再用 `GROUPBY` 按位置对各表对应行求和。

适用前提：所有分表行数、字段顺序及产品行位置完全一致。如果各表产品顺序不同，应改为按产品代码分组，而不是按行号分组。

## 3. 分组循环

辅助索引位置：`Sheet1!A2`，溢出范围 `A2:A16801`

```excel
=SEQUENCE(C1)
```

每 1000 行分批拆分，位置：`Sheet1!D2`，溢出范围 `D2:I16801`

```excel
=LET(arr,B2:B16801,n,1000,r,INT(ROWS(arr)/n),fx,LAMBDA(x,TEXTSPLIT(TEXTJOIN(";",,TAKE(DROP(arr,x*n),n)),",",";")),REDUCE(fx(0),SEQUENCE(r),LAMBDA(x,y,VSTACK(x,fx(y)))))
```

每 1200 行分批拆分，位置：`Sheet1!K2`，溢出范围 `K2:P16801`

```excel
=LET(arr,B2:B16801,n,1200,r,ROUNDUP(ROWS(arr)/n,0)-1,fx,LAMBDA(x,TEXTSPLIT(TEXTJOIN(";",,TAKE(DROP(arr,x*n),n)),",",";")),REDUCE(fx(0),SEQUENCE(r),LAMBDA(x,y,VSTACK(x,fx(y)))))
```

解题结构：大列文本不能一次稳定拆分时，先用 `DROP + TAKE` 按固定行数取块；每块通过 `TEXTJOIN + TEXTSPLIT` 转成二维数组；最后由 `REDUCE + VSTACK` 合并所有批次。

可泛化方法：大数据任务先分块，再对每块执行同一变换，最后累积合并。块数必须覆盖最后一个不足整块的尾批次。

## 4. 根据次数重复显示多行

位置：`Sheet2!J2`，溢出范围 `J2:Q1521`

```excel
=LET(data,A2:H240,arr,CHOOSECOLS(data,8),rIndex,SEQUENCE(ROWS(data)),brr,IF(arr>=SEQUENCE(,MAX(arr)),rIndex,x),CHOOSEROWS(data,TOCOL(brr,2)))
```

解题结构：把每行重复次数与横向 `1..最大次数` 比较，广播生成“行号矩阵”；不满足次数的位置放占位值；`TOCOL(...,2)` 忽略占位并展平有效行号；`CHOOSEROWS` 按行号重复抽取整行。

可泛化方法：次数向量 -> 广播掩码 -> 有效索引流 -> 按索引展开原记录。

注意：原公式中的 `x` 未定义。正式知识需要按目标环境明确替换为可被 `TOCOL(...,2)` 忽略的错误值，例如 `NA()`，不能原样当作跨环境通用写法。

## 5. 户主与家庭成员归属

方案一位置：`数据源!D2`，溢出范围 `D2:E1004`

```excel
=LET(data,FILTER(A2:B1310,A2:A1310<>B2:B1310),rowCount,SEQUENCE(ROWS(data)),arr,CHOOSECOLS(data,1),brr,CHOOSECOLS(data,2),户主,INDEX(arr,XMATCH(rowCount,IF(arr="","",rowCount),-1)),成员,IF(brr="",arr,brr),HSTACK(户主,成员))
```

方案二位置：`数据源!G2`，溢出范围 `G2:H1004`

```excel
=LET(data,FILTER(A2:B1310,A2:A1310<>B2:B1310),arr,CHOOSECOLS(data,1),brr,CHOOSECOLS(data,2),户主,SCAN("",arr,LAMBDA(x,y,IF(y="",x,y))),成员,IF(brr=0,arr,brr),HSTACK(户主,成员))
```

解题结构：稀疏的户主列需要向下继承。方案一把非空户主所在行号作为断点，用 `XMATCH(...,-1)` 找当前位置之前最近断点；方案二用 `SCAN` 保存“最近一个非空户主”。最后规范成员列并 `HSTACK` 输出。

共同方法：层级首行值 + 下方空白明细 -> 最近有效值向下填充 -> 形成标准化二维记录。

## 6. 降序/区间匹配

位置：`Sheet10 (2)!H2`，溢出范围 `H2:I13`

```excel
=LET(arr,--TEXTAFTER(A2:A5,"-",,,,A2:A5),brr,SEQUENCE(MAX(arr)-MIN(arr)+1,,MIN(arr)),HSTACK(TEXT(brr,"000"),XLOOKUP(brr,arr,B2:B5,,1)))
```

解题结构：先从单值或区间文本提取每段上界；生成最小到最大值的完整序列；用 `XLOOKUP` 的“精确或下一个较大项”把每个值映射到对应区间属性。

可泛化方法：区间上界表 -> 连续键序列 -> 上界阶梯匹配。

## 7. 利用 INDEX 特性统计数据

位置：`统计!A5`，溢出范围 `A5:I21`

```excel
=LET(数据源,汇总数据[[成交项目]:[进度]],起始日期,">="&B1,结束日期,"<="&B2,fx,LAMBDA(c,INDEX(数据源,,c)),func,LAMBDA(arr,INDEX(arr,,1)-INDEX(arr,,2)-INDEX(arr,,3)),项目,SORT(UNIQUE(fx(1))),数量统计,COUNTIFS(fx(1),项目,fx(4),{"*","*退单*","*结单*"},fx(2),起始日期,fx(2),结束日期),在库套数,func(数量统计),金额统计,SUMIFS(fx(3),fx(1),项目,fx(4),{"*","*退单*","*结单*"},fx(2),起始日期,fx(2),结束日期),在库业绩,func(金额统计),HSTACK(项目,数量统计,在库套数,金额统计,在库业绩))
```

解题结构：用 `INDEX(data,,列号)` 封装列访问器；唯一项目数组作为多条件统计条件；状态数组常量让 `COUNTIFS/SUMIFS` 一次返回多列结果；再用第二个 `LAMBDA` 计算“总量 - 退单 - 结单”，最后横向拼接。

可泛化方法：维度数组与状态数组同时广播，单次生成“每个项目 × 多种状态”的统计矩阵。

## 8. 两组数据生成全部两列组合

位置：`Sheet2!E1`，溢出范围 `E1:F20`

```excel
=TEXTSPLIT(TEXTJOIN(";",,A1:A4&","&TOROW(B1:B5)),",",";")
```

解题结构：纵向数组与 `TOROW` 后的横向数组直接拼接，利用数组广播得到笛卡尔积文本矩阵；`TEXTJOIN` 展平，`TEXTSPLIT` 再恢复为两列。

可泛化方法：纵向向量 × 横向向量 -> 广播组合 -> 展平 -> 按字段和记录分隔符还原。

## 9. 内容重排

位置：`Sheet1!C9`，溢出范围 `C9:H15`

```excel
=LET(a,DROP(REDUCE(0,A1:A13,LAMBDA(x,y,VSTACK(x,IF(LEN(y)>13,HSTACK(TAKE(x,-1,1),REGEXP(y,"[\d\-.]+")),HSTACK(REGEXP(y,"/\K.*"),""))))),1),FILTER(a,CHOOSECOLS(a,2)>""))
```

解题结构：输入是“标题行 + 明细行”交替出现的非标准文本。`REDUCE` 逐行扫描；标题行提取主键，明细行通过 `TAKE(x,-1,1)` 继承上一主键并正则拆字段；`VSTACK` 累积成表，最后过滤无明细行。

共同方法：异构行识别 -> 状态继承 -> 字段解析 -> 累积为标准记录表。

注意：`REGEXP` 是 WPS 语义，Office Excel 需要按实际版本改为对应正则函数或文本拆分方案。

## 10. 索引转置

位置：`Sheet1!C1`，溢出范围 `C1:E277`

```excel
=LET(data,A1:A739,r,MAP(data,LAMBDA(x,N(COUNT(--MID(x,SEQUENCE(LEN(x)),1))>7))),rIndex,SCAN(,r,SUM),cIndex,SCAN(,r,LAMBDA(x,y,IF(y=0,x+1,1))),DROP(PIVOTBY(rIndex,cIndex,data,SINGLE,0,0,,0),1,1))
```

解题结构：先逐项统计数字字符，识别每组起始记录；第一次 `SCAN` 对起始标记累计，生成组号；第二次 `SCAN` 在起始处重置、组内递增，生成列号；`PIVOTBY` 按“组号 × 组内位置”把一列转成多列。

共同方法：识别分组边界 -> 生成组号 -> 生成组内序号 -> 透视重塑。

## 11. 提取每行不同位置的目标列

位置：`提取数据!H1`，溢出范围 `H1:H15`

```excel
=LET(data,总表!A1:CR15,arr,BYROW(data,LAMBDA(x,IFNA(MATCH("skuId",x,0),MATCH("skuId",OFFSET(x,1,0),0)))),colIndex,SCAN(,arr,LAMBDA(x,y,IFNA(y,x))),INDEX(data,SEQUENCE(ROWS(data)),colIndex))
```

解题结构：每个数据块的 `skuId` 表头位于不同列。`BYROW` 查找当前行或下一行中的目标表头位置；`SCAN` 把最近识别到的列号向下传递；最后用“行序号数组 + 每行列号数组”让 `INDEX` 对每行取不同列。

共同方法：动态识别列位置 -> 状态向下继承 -> 逐行使用不同列索引。

## 12. 统计每月缺勤人数

真实日期列标题方案，位置：`启停餐明细!G2`，溢出范围 `G2:L16`

```excel
=LET(班级,明细表[班级],起始日期,明细表[停餐日期],结束日期,明细表[启餐日期],所有日期,SEQUENCE(,MAX(结束日期)-MIN(起始日期)+1,MIN(起始日期)),缺勤日期,(所有日期>=起始日期)*(所有日期<=结束日期),fx,LAMBDA(x,TOCOL(IF(缺勤日期,x,NA()),2)),PIVOTBY(fx(班级),EOMONTH(--fx(所有日期),-1)+1,fx(班级),COUNTA))
```

文本月份标题方案，位置：`启停餐明细!A51`，溢出范围 `A51:F65`

```excel
=LET(班级,明细表[班级],停餐日期,明细表[停餐日期],截止日期,明细表[启餐日期],所有日期,SEQUENCE(,MAX(截止日期)-MIN(停餐日期)+1,MIN(停餐日期)),缺勤日期,(所有日期>=停餐日期)*(所有日期<=截止日期),fx,LAMBDA(arr,TOCOL(IF(缺勤日期,arr,x),2)),PIVOTBY(fx(班级),TEXT(fx(所有日期),"mm月"),fx(班级),COUNTA))
```

解题结构：人员记录纵向数组与完整日期横向数组广播，形成“人员 × 日期”的区间布尔矩阵；把班级和日期按掩码展平成长表；再按班级和月份透视计数。

共同方法：区间记录 -> 展开到每日粒度 -> 掩码筛选 -> 长表化 -> 透视统计。

注意：第二个原公式中的 `x` 未定义；正式入库应采用第一种 `NA()` 占位策略或明确空值过滤规则。

## 13. 多条件模糊查询

位置：`查询!A7`，溢出范围 `A7:AE16`

```excel
=LET(data,投保明细!B3:AF90,rs,SEQUENCE(ROWS(data))^0,fx,LAMBDA(c,CHOOSECOLS(data,c)),FILTER(data,IF(C1="",rs,fx(12)=C1)*ISNUMBER(FIND(C2,fx(-3)))*IF(C3="",rs,fx(-2)=C3)*IF(C4="",rs,fx(9)>=C4)*IF(E4="",rs,fx(9)<=E4)))
```

解题结构：`SEQUENCE(...)^0` 生成全 1 条件向量；每个可选条件为空时返回全 1，否则返回实际判断；`FIND + ISNUMBER` 实现包含式模糊匹配；所有条件数组相乘后交给 `FILTER`。

共同方法：可选筛选条件 = “空输入不过滤，非空输入才约束”；模糊、精确和区间条件可以统一组合为布尔乘积。

## 14. 重复多列数据/多维组合

条件筛选价格表，位置：`价格表!B6`，溢出范围 `B6:G12`

```excel
=FILTER(配置!D2:I999,(配置!A2:A999=B2)*(配置!B2:B999=B3))
```

多维组合展开，位置：`配置!O2`，溢出范围 `O2:Q141`

```excel
=LET(arr,L2:M8,brr,REDUCE(K2:K5,SEQUENCE(COLUMNS(arr)),LAMBDA(x,y,TOCOL(x&","&TOROW(INDEX(arr,,y),1)))),TEXTSPLIT(TEXTJOIN(";",,brr),",",";"))
```

解题结构：`FILTER` 根据两个维度返回匹配的整块价格矩阵；组合公式则以第一维为初始集合，`REDUCE` 逐列加入新维度，每轮通过纵横广播生成新的笛卡尔积，最后还原成多列。

共同方法：多维组合不是一次写死，而是把“加入一个维度”定义为可重复的归约步骤。

## 15. 组内排名

位置：`Sheet2!C2`，溢出范围 `C2:C13`

```excel
=LET(arr,A2:A13,brr,B2:B13,MAP(arr,brr,LAMBDA(x,y,SUM(1*(UNIQUE(FILTER(brr,arr=x))>=y)))))
```

解题结构：`MAP` 同时遍历当前组和当前成绩；`FILTER` 取当前组全部成绩；`UNIQUE` 去重后统计大于等于当前成绩的档位数，实现组内降序密集排名，同分同名次且下一名不跳号。

共同方法：逐记录计算时，先按当前记录的分组键取得组内子集，再对当前值执行组内统计。

## 从 15 个案例归纳出的共同解题框架

### 1. 先判断输出形状

- 单值：模板替换。
- 等长一列：组内排名、逐行动态取列。
- 变长明细：按次数重复、区间展开。
- 二维矩阵：跨表汇总、透视统计、长宽转换。
- 笛卡尔积：两组或多维组合。

### 2. 再生成“索引或状态”，最后取值

这些案例很少直接拼最终结果，通常先产生中间控制数组：

- `SEQUENCE` 生成位置或完整日期。
- 广播比较生成掩码或重复行索引。
- `SCAN` 生成最近有效值、组号或组内序号。
- `XMATCH/MATCH` 生成动态位置。
- 最后由 `INDEX/CHOOSEROWS/FILTER/PIVOTBY` 取值或重塑。

### 3. 三类核心迭代函数职责明确

- `MAP`：每项独立计算，例如组内排名、识别分组起点。
- `SCAN`：当前结果依赖前序状态，例如向下填充、组号和列号。
- `REDUCE`：结果尺寸逐轮增长，例如分块合并、文本重排、多维组合。

### 4. 复杂数组公式的稳定结构

`LET 定义输入 -> 构造索引/掩码 -> 执行局部变换 -> 展平或累积 -> 重塑 -> HSTACK/VSTACK 输出`

### 5. 正式知识库不应照抄案例地址

每条知识应保存：输入形状、目标形状、变换链、函数职责、适用前提、边界条件和验证方式。案例中的固定范围、工作表名、未定义占位符及 WPS 专用函数只作为来源证据，不能直接成为通用答案。
